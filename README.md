# AI Content Engine Lite - Public Demo Version

This is the free evaluation version of AI Content Engine with limited features for demo purposes.

## Contents

```
Lite_Public/
├── src/                        # Source code
│   ├── ai_content_engine_lite.py   # Main CLI application
│   └── modes/                      # Content generation modules
│       ├── blog.py                 # Blog post generator
│       └── tweet_thread.py         # Tweet thread generator
├── LICENSE.md                  # Demo License Agreement
├── UPGRADE_TO_FULL.md          # Upgrade information
└── README.md                   # This file
```

## What's Included

### `/src`
A subset of the complete source code:
- Blog post generation (template-based)
- Tweet thread generation (template-based)
- Basic CLI interface

## Quick Start

```bash
# Navigate to source directory
cd src

# Generate a blog post
python ai_content_engine_lite.py --mode blog --topic "your topic here"

# Generate a tweet thread
python ai_content_engine_lite.py --mode tweet_thread --topic "your topic here"

# Interactive mode
python ai_content_engine_lite.py --interactive
```
